## Installation

```sh
yarn add @urql/vue graphql
# or
npm install --save @urql/vue graphql
```

> **Note:** `@urql/vue` has a peer dependency on `vue@^3.0.0` (Not v2) and doesn't currently plan to
> be backwards compatible to Vue 2.
